Approach

I’ve always been interested in web design but have never been brave enough to try web Development. Following a brief set by the staff at Nottingham Trent, I set out to learn as much as I could about front-end development (HTML, CSS, JavaScript) in the hope to produce a final outcome documenting my learning.

Outcome

My final outcome was a portfolio style website that documented all my learning. It also had links to my personal pages and information about how I found the projects, the things I had learnt etc. By the end of this project I had taught myself HTML, CSS and JavaScript fundamentals, along with other tasks such as version control.

Tools

HTML • CSS/SCSS • JavaScript • Responsive design • Version control with git • Adobe Suite • Figma
