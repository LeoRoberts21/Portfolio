Dough Mother

UX/UI • Front-End • Branding

A self-initated university brief. I designed and coded a website for a local small business. This was done with the goal of improving my front-end skills.

Repo

Live Preview

Sketchbook

Approach

After researching lots of local business’s, I decided that DoughMother, a bakery in Beaston would benefit from a site the most. They are a bakery that I felt could be reaching more customers if they had an online presence like their competition, some of whom have over 10k followers on Instagram.

Outcome

Using html, Scss and JavaScript, I designed and coded a functional website, responsive to various screen sizes. The website features information, pictures, recipes, and a mailing list for customers to get access to offers. I also coded a working cart that lets the user add items to make online orders.

Tools

HTML • CSS/SCSS • JavaScript • Responsive design • Version control with git • Adobe Suite • Figma
